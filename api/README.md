# Simple Express API with OpenAI

**Require: Node >= 16**

## Install Dependencies & Setup Environment  

```sh
cd ./api
npm i
cp ./.env.example .env
```

### OpenAI API Configuration

Configure OpenAI API settings in .env:

```
OPENAI_API_KEY=
OPENAI_VOICE_MODEL=gpt-realtime
```

## Start API Application

```
npm run dev
```
